// Lumora Skincare Landing Page
A modern, fully responsive e-commerce landing page for a premium skincare brand. Built with a focus on clean architecture, performance, and high-end aesthetics.

// Features
Modern UI/UX: Minimalist design with a "soft-aesthetic" color palette.

Mobile-First: Fully responsive layout using CSS Grid and Flexbox.

BEM Methodology: Organized, scalable CSS following the Block Element Modifier naming convention.

Clean Code: Highly commented, beginner-friendly JavaScript for interactive elements.

Performance: Lightweight and optimized for fast loading across all devices.
